# TIL: Rust Basics

**Date:** 2026-03-08

## What I Learned

I was working on a Rustlings exercise and didn't understand the error that the compiler was giving me because it said that something was wrong with one of my variables, but not the other variable which was just like the first one. I found out that Rust will not necessarily report all errors at once. It will report one error and then after you re-compile, then other errors will pop up.

## Why It Matters

This will help me with future error troubleshooting because now I know to just fix the error that the compiler is bringing up and then I can see if it finds any other errors once that code has been fixed.
