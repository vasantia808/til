# TIL: Markdown Basics

**Date:** 2026-03-07

## What I Learned

Markdown is a lightweight formatting language used to create formatted text 
using plain text syntax. It is widely used for documentation, README files, 
and notes on GitHub.

## Basic Syntax

- `#` creates a heading (more # symbols = smaller heading)
- `**text**` makes text **bold**
- `*text*` makes text *italic*
- `-` creates a bullet point
- Backticks create `inline code`

## Why It Matters

Markdown is the standard format for GitHub documentation, writeups, and 
technical notes. Learning it is essential for maintaining a visible, 
professional presence on GitHub.
